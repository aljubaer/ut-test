# ut-test

## Title 
Sample Node.js api service for authentication and authorization

## Setup
Follow the steps:
- `git clone `
- `cd ut-test`
- `npm install`
- `npm run dev`

## Dependencies
- node^16

## Features
- Customer login: Provide `"role": "customer"`
- Admin login: Provide `"role": "customer"`
- Customer hello world
- Admin hello world

## Documentation
Navigate `/api-docs` for swagger documentation
